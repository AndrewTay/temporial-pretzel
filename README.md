# Temporial

**Browser-Native 3D Previs & Structural Render Bridge for Generative Video**

A zero-install, multiplayer 3D previs workspace designed to emit *ground-truth* structural conditioning maps (depth, optical flow, segmentation, camera tracks) that force generative video diffusion models to respect an agency's spatial composition — turning "composition and asset consistency drift" from a budget-level problem into a controllable parameter.

A co-founder invitation site.

## View
Open `index.html` (or `temporial-cinematic.html`) in a browser. The page is self-contained.

## Structure
- `index.html` / `temporial-cinematic.html` — the site
- `assets/` — videos, images, 3D models, and the scroll-background video
